# DigiTools

> A modern digital tools marketplace built with React and Vite.

## About

DigiTools is a clean and responsive web app for exploring premium digital products, managing a cart, and viewing pricing plans in a smooth user experience.

## Technology Used

- React
- Vite
- Tailwind CSS
- daisyUI
- React Toastify

## Features

1. Product browsing with dynamic product cards and responsive layouts.
2. Cart management with toast notifications for add, remove, and checkout actions.
3. Pricing and workflow sections designed for a polished landing page experience.
